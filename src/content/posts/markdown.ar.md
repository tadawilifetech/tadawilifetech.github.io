# عنوان H1

يتم الفصل بين الفقرات بسطر فارغ.

هذه فقرة ثانية. يمكن كتابة _المائل_ و**العريض** و`النص الأحادي` بسهولة. ويمكن إنشاء قوائم مثل:

- العنصر الأول
- العنصر الثاني
- العنصر الثالث

> تكتب الاقتباسات بهذا الشكل.
>
> ويمكن أن تمتد عبر عدة فقرات إذا أردت.

## عنوان H2

وهذه قائمة مرقمة:

1. العنصر الأول
2. العنصر الثاني
3. العنصر الثالث

إليك مثالاً على كتلة كود:

```
define foobar() {
    print "Welcome to flavor country!";
}
```

كما يمكنك تمييز الكود بلغة معينة:

```python
import time
for i in range(10):
    time.sleep(0.5)
    print(i)
```
