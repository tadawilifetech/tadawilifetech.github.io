# عنوان H1

پاراگراف‌ها با یک خط خالی از هم جدا می‌شوند.

این پاراگراف دوم است. می‌توانید متن _مورب_، **پررنگ** و `تک‌عرض` بنویسید. فهرست‌ها هم به این شکل هستند:

- مورد اول
- مورد دوم
- مورد سوم

> نقل‌قول‌ها این‌گونه نوشته می‌شوند.
>
> و می‌توانند در چند پاراگراف ادامه پیدا کنند.

## عنوان H2

یک فهرست شماره‌دار:

1. مورد اول
2. مورد دوم
3. مورد سوم

نمونه‌ای از بلوک کد:

```
define foobar() {
    print "Welcome to flavor country!";
}
```

همچنین می‌توانید زبان کد را مشخص کنید:

```python
import time
for i in range(10):
    time.sleep(0.5)
    print(i)
```

### عنوان H3

فهرست تو در تو هم پشتیبانی می‌شود:

1. این مواد را آماده کنید:

    - هویج
    - کرفس
    - عدس

2. کمی آب بجوشانید.

3. همه چیز را داخل قابلمه بریزید و مراحل را دنبال کنید.

اینجا یک [لینک](http://foo.bar) و یک [ارجاع داخلی](#عنوان-h2) دارید.

جدول‌ها هم می‌توانند به این شکل باشند:

| سایز | جنس | رنگ |
|------|------|------|
| 9 | چرم | قهوه‌ای |
| 10 | کنف | طبیعی |
| 11 | شیشه | شفاف |

فرمول درون‌خطی به این شکل است: $\omega = d\phi / dt$.

$$I = \int \rho R^{2} dV$$
